font-size

dèfinition et utilisation

Ces deux symboles sont utilisés tels quels dans la définition d'une valeur.… …La seule différence est l'emplacement de leur définition : dans ce cas, la définition est généralement très proche de la… …définition de la propriété qui les utilise.…

exemple

/* Le paragraphe sera écrit avec une grande */ /* fonte. */ p { font-size: xx-large; } /* La taille du texte des h1 sera 2.5 fois plus */ /* grande que le texte autour. */ h1 { font-size: 250%; } /* Le texte contenu dans des éléments span */ /* mesurera 16px */ span { font-size: 16px; }
